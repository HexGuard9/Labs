These are my Windows Server lab. I used this to practice AD, GPO, DHCP/DNS and joining clients to domain


## Skills Demonstrated

- **Group Policy Objects (GPO)**: policies ✅  
- **DHCP & DNS**: basic configuration and IP management ✅  
- **Joining Windows clients to Domain Controller** ✅
